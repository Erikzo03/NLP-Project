\section{Dataset}

We use the Universal NER benchmark \citep{mayhew2024universal} for English, Norwegian, and Danish. The data is sentence-level and token-aligned in IOB2 format, where each token carries a BIO label (B-TYPE, I-TYPE, or O). The entity types are PER (person), LOC (location), and ORG (organization). MISC entities appear in a second annotation column of the Danish file but are absent from the primary label column that models read and are evaluated on, so MISC F1 is zero for all systems by construction --- not a model failure but a data-format artifact we discuss in Section~\ref{sec:analysis}.

The three languages play different roles in the experiments. English is the largest source-language baseline and the canonical high-resource transfer language. Norwegian is the North Germanic alternative we compare against English. Danish is the target language whose labeled data we treat as scarce in the transfer experiments.

The dataset characteristics are not symmetric. English has roughly five times more training sentences than Danish or Norwegian, which gives it a data-volume advantage in the source-language baselines that is separate from any linguistic-proximity effect. Danish and Norwegian are matched in size, so the Danish data-budget experiments (10\%, 25\%, 50\%) have a clean interpretation.

The label inventory is shared across languages in the primary annotation column, so performance differences reflect language transfer rather than annotation-schema mismatch. We verified this consistency before training using a label-inventory check across all three datasets.
