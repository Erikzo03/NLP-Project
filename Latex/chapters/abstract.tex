\begin{abstract}
We study cross-lingual named entity recognition (NER) transfer to Danish using multilingual transformer encoders (mBERT and XLM-R). The report follows the project proposal and expands it into a full ACL-style paper with detailed experiments, linguistic distance analysis, and error inspection. Our main finding is that Norwegian shows a small, directionally consistent advantage over English as a zero-shot transfer source for Danish: 86.5 versus 85.5 strict F1 on the development set. A paired bootstrap test confirms, however, that this gap is not statistically significant at any conventional level ($p=0.57$), which means multi-seed replication is needed before a strong claim can be made. When fine-tuning on Danish data, an English mBERT model matches or outperforms Norwegian transfer models, and Danish-only supervision remains the strongest option once enough labeled data is available. The directional advantage of Norwegian is consistent with a typological distance analysis showing Norwegian is 5.7$\times$ closer to Danish than English across WALS syntax features. We also report per-entity-type results, a data-efficiency curve, and identify one prediction file as a misaligned artifact.
\end{abstract}
