\begin{abstract}
Cross-lingual named entity recognition (NER), the task of identifying person, location, and organization spans in text, offers a practical solution for low-resource languages like Danish, where labeled data is scarce: train on a resource-rich source language and transfer without target labels. We ask whether Norwegian, a structurally closer North Germanic language, is a more effective transfer source for Danish NER than English. Using mBERT and XLM-R across three random seeds, Norwegian leads in zero-shot transfer: +1.86 F1 for XLM-R ($p=0.10$) and +3.36 F1 for mBERT ($p=0.046$, significant at $\alpha=0.05$). Multi-seed replication proved essential: a single seed had favored Norwegian XLM-R on dev, only to reverse when averaged across seeds, confirming that source-language claims require held-out test evaluation. When Danish fine-tuning is available, English mBERT achieves the best result (89.4 dev strict F1), showing that corpus size can outweigh typological proximity once target supervision is available.\footnote{Code: \url{https://github.com/Erikzo03/NLP-Project}}
\end{abstract}
